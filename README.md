# propaganda_nuke

A browser extension for resisting emotional capture.

propaganda_nuke blocks, blurs, and inserts friction between your attention and systems designed to farm fear, outrage, and helplessness.

---

## What it does

* Removes “Top stories” and news modules from Google
* Blurs or hides manipulative search results
* Blocks selected news and social domains
* Intercepts clicks on known attention traps
* Forces a pause before opening high-risk content
* Tracks how many clicks you didn’t make

---

## Why this exists

Modern feeds and headlines are not neutral.

They are optimized for:

* emotional spike
* compulsive engagement
* outrage loops
* anxiety reinforcement

This extension does not try to decide what is *true*.

It targets **what is engineered to hijack your attention**.

---

## Core idea

Don’t fight content.

Break the loop between:

> stimulus → impulse → click → emotional state

Insert friction.

Give yourself a second to choose.

---

## Features

* **Hard block mode**
  Redirects blocked domains to an intervention screen

* **Friction mode**
  Countdown before opening known manipulation-heavy links

* **Keyword detection**
  Flags headlines using outrage/anxiety language patterns

* **Google filtering**
  Hides results from blocked domains + suppresses news modules

* **Local-only**
  No accounts, no tracking, no external servers

---

## Install (dev mode)

1. Clone the repo
2. Go to `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select the project folder

---

## Configuration

Open the extension → Settings

You can:

* edit blocked domains
* customize keyword triggers
* adjust countdown delay
* toggle strict / friction modes

---

## Philosophy

This is not a censorship tool.

It is a **self-defense tool**.

You decide what gets blocked.
You can override anything.
But you don’t get pulled in automatically.
